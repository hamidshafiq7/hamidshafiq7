## Deployment

This repo IS the GitHub Pages site. Push to `main` branch — GitHub Pages serves `index.html` automatically at the root domain.

1. Create repo named exactly `hamidshafiq1.github.io`
2. Push all files to `main` branch
3. Go to **Settings → Pages → Source: Deploy from branch → main / root**
4. Site goes live at `https://hamidshafiq1.github.io` within ~60 seconds

## Adding Certificate Images

Place all cert images inside `assets/certs/`. Filenames must match exactly what is referenced in `index.html`.
